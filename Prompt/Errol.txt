1. Prompt สำหรับ "ใบรายละเอียดการเก็บเงิน" (หน้า 1)
หน้านี้เน้นตารางรายการที่สะอาดตา และยอดรวมท้ายตาราง

รับบทเป็น Frontend Developer ช่วยเขียนโค้ด HTML และ CSS สำหรับทำ "ใบแจ้งรายละเอียดการเก็บเงิน (School Fee Receipt)" เพื่อนำไปปริ้นลงกระดาษ A4

**ความต้องการ (Requirements):**
1. **การตั้งค่าหน้ากระดาษ:** กำหนด CSS `@page { size: A4; margin: 0; }` และให้ body margin เป็น 0
2. **โครงสร้าง (Layout):**
   - **ส่วนหัว:** จัดกึ่งกลาง ใส่ข้อความ "วิทยาลัยอาชีวศึกษานครปฐม", "รายละเอียดการเก็บเงิน ปวช.1 ทุกแผนกวิชา", "ภาคเรียน 2/2568"
   - **ตารางรายการ:** สร้างตารางที่มีเส้นขอบชัดเจน (Bordered Table) มีคอลัมน์: "ลำดับที่", "รายการ", "จำนวนเงิน"
   - **ข้อมูลสมมติ:** ใส่ข้อมูลตัวอย่าง 3-4 แถว เช่น "ค่ากิจกรรมกีฬาสี 250", "ค่าบำรุงห้องสมุด 100"
   - **ส่วนท้ายตาราง:** แสดงยอดรวมเป็นตัวอักษร "(สามร้อยห้าสิบบาทถ้วน)" และตัวเลข "350" ตัวหนา
   - **หมายเหตุ:** ใส่ข้อความตัวเล็กด้านล่าง "*หมายเหตุ ค่าธรรมเนียมจ่ายผ่านธนาคาร 10 บาท"
3. **สไตล์ (Styling):**
   - เขียน CSS ใน tag `<style>` (ไม่ต้องแยกไฟล์)
   - ใช้ฟอนต์ภาษาไทย 'Sarabun' จาก Google Fonts
   - เน้นดีไซน์ขาว-ดำ (Black & White) สะอาดตาสำหรับการพิมพ์

2. Prompt สำหรับ "ใบแจ้งการชำระเงิน" (หน้า 2 - Pay-in Slip)
อันนี้สำคัญสุดครับ ต้องสั่งให้แบ่งครึ่งกระดาษให้เป๊ะ
หน้านี้ซับซ้อนสุด ต้องเน้นเรื่องการแบ่งครึ่งกระดาษ (Split Layout) และกล่อง Ref1/Ref2

รับบทเป็น Frontend Developer ช่วยเขียนโค้ด HTML และ CSS แบบละเอียดสำหรับทำ "ใบแจ้งการชำระเงินธนาคาร (Bank Pay-in Slip)"

**ความต้องการ (Requirements):**
1. **การจัดหน้า (Layout Strategy):**
   - ขนาดกระดาษ A4 แนวตั้ง
   - **สำคัญมาก:** ต้องแบ่งหน้ากระดาษออกเป็น 2 ส่วนเท่ากันในแนวตั้ง (บน 50% และ ล่าง 50%)
   - **ส่วนบน:** เขียนกำกับว่า "ส่วนที่ 1 (สำหรับนักเรียน)"
   - **ส่วนล่าง:** เขียนกำกับว่า "ส่วนที่ 2 (สำหรับธนาคาร)"
   - คั่นกลางด้วยเส้นประ (Dashed Line) สำหรับฉีก

2. **เนื้อหา (ทำเหมือนกันทั้งส่วนบนและล่าง):**
   - **Header:** ด้านซ้ายใส่ Logo และชื่อ "วิทยาลัยอาชีวศึกษานครปฐม", ด้านขวาใส่ "Company Code: 81245"
   - **ข้อมูลนักเรียน:**
     - ชื่อ-นามสกุล
     - **ช่อง Ref (สำคัญ):** สร้างกล่องสี่เหลี่ยมเล็กๆ เรียงกันสำหรับกรอก "Ref.1 (รหัสประจำตัว)" และ "Ref.2 (รหัสกลุ่ม)" ให้เหมือนช่องกรอกรหัสในใบแจ้งหนี้จริง
   - **ตารางยอดเงิน:** คอลัมน์ "ลำดับ", "รายการ", "จำนวนเงิน" (ใส่ยอดค่าเทอม 2,500 และ ค่าอื่นๆ 750)
   - **ส่วนท้าย:** - ยอดเงินรวม (ตัวอักษร และ ตัวเลข)
     - โลโก้หรือชื่อธนาคาร "ธนาคารกรุงไทย"
     - ช่องสำหรับเจ้าหน้าที่ธนาคารเซ็นชื่อ และประทับตรา

3. **เทคนิค CSS:**
   - ใช้ `box-sizing: border-box` เพื่อคำนวณขนาดให้เป๊ะ
   - ความสูงรวมต้องไม่เกิน 297mm (A4) ห้ามล้นไปหน้า 2
   - ใช้ Flexbox จัดเรียงกล่อง Ref1/Ref2 ให้สวยงาม
   - ใช้ฟอนต์ 'Sarabun'

3. Prompt สำหรับ "บัตรลงทะเบียนรายวิชา" (หน้า 3)
สำหรับหน้าตารางเรียน

รับบทเป็น Frontend Developer ช่วยเขียนโค้ด HTML และ CSS สำหรับ "บัตรลงทะเบียนรายวิชา (Student Course Registration Card)"

**ความต้องการ (Requirements):**
1. **Header:** - ข้อความกึ่งกลาง "วิทยาลัยอาชีวศึกษานครปฐม", "บัตรลงทะเบียนรายวิชา", "ภาคเรียนที่ 2/2568"
   - ข้อมูลส่วนตัว: ชื่อนักเรียน, รหัสนักศึกษา, สาขาวิชา, ระดับชั้น (ปวช.1)

2. **ตารางรายวิชา (Main Table):**
   - คอลัมน์: "ลำดับ", "รหัสวิชา", "ชื่อวิชา", "ท (ทฤษฎี)", "ป (ปฏิบัติ)", "นก. (หน่วยกิต)"
   - ข้อมูลสมมติ: ใส่มา 5 วิชา (เช่น CS101 การพัฒนาเว็บ, CS102 คณิตศาสตร์)
   - **แถวสรุป (Footer Row):** รวมจำนวนชั่วโมงทฤษฎี, ปฏิบัติ และหน่วยกิตรวมไว้ท้ายตาราง

3. **ส่วนสรุปและลายเซ็น (Footer Layout):**
   - แบ่งเป็น 2 ฝั่ง
   - **ฝั่งซ้าย:** ตารางสรุปยอดเงินเล็กๆ (ค่าบำรุงการศึกษา 350 บาท)
   - **ฝั่งขวา:** ช่องสำหรับลงชื่อ 3 ช่อง ได้แก่ "นักศึกษา", "ครูที่ปรึกษา", "งานทะเบียน" โดยมีเส้นจุดไข่ปลา (dotted line) ให้เซ็น
   - ข้อความด้านล่างสุด: "*ชำระเงินได้ที่ธนาคารกรุงไทย..."

4. **สไตล์:**
   - ใช้ตารางแบบ `border-collapse: collapse` เส้นขอบสีดำบางๆ
   - ใช้ฟอนต์ 'Sarabun'
